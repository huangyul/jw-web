## 几个常见问题

###### 1. 为什么要 diff

本质上就是为了性能！！

MVVM 的本质： **数据模型** -> **virtual dom** -> **视图(dom)**，通过虚拟 dom 去关联数据模型和视图

虚拟 Dom 本质就是一个对象

```js
{
  type: 'div',
  props: {...}
}
```

注意：真实 dom 操作并不慢，不一定真实 dom 就是慢！！

但是真实操作 dom，会容易引起重绘和回流
